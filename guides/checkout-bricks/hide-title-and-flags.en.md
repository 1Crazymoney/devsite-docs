> Hide title and flags

| Brick  | Card Payment Form  |
| --- | --- |
| Customization moment  | When rendering the brick  |
| Property  | customization.hideFormTitle  |
| Type  | Boolean  |
| Comments  | When **true**, hides the title line and accepted flags.  |

```javascript
const settings = {
   ...,
   customization: {
       visual: {
           hideFormTitle: true
       }
   }
}
```
